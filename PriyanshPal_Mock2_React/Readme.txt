SETUP INSTRUCTIONS
------------------
1. Extract the ZIP file
2. Open terminal in the project folder
3. Install dependencies:
   npm install

4. Install additional packages:
   npm install react-router-dom axios formik yup bootstrap redux react-redux prop-types

5. Start JSON Server (in separate terminal):
   json-server --watch db.json --port 3001

6. Start React Application:
   npm start

7. Application runs on: http://localhost:3000

PAGES AVAILABLE
---------------
1. Home (/) - Landing page
2. Packages (/packages) - Browse travel packages
3. Book Now (/booking) - Booking form
4. Dashboard (/dashboard) - View and manage bookings
5. Contact (/contact) - Contact information
